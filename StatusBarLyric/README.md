# 🎵 状态栏歌词解决方案：词幕 (Lyricon)

## 介绍
截至 **2026年3月13日**，本软件（QZMusic）支持以下三种状态栏歌词显示方案：
1. **AviumUI** 类原生定制
2. **Flyme** 系统定制
3. **词幕 (Lyricon)** LSPosed 模块

> **💡 为何选择词幕？**
> 对于非 Flyme 或非 AviumUI 系统的用户，**词幕**是实现状态栏歌词显示的目前已知通用解决方案中最完善&美观的一个。

## ✨ 核心特性
基于 [Lyricon GitHub 仓库](https://github.com/tomakino/lyricon)，该模块提供极致的歌词体验：
*   **🎤 多形态展示**：支持逐字歌词、双语翻译及对唱模式，打造沉浸式听歌体验。
*   **🧩 模块化扩展**：通过独立插件机制无限扩展歌词源，轻松适配各类播放器。
*   **🎨 深度自定义**：支持字体、Logo、坐标偏移及动态效果调整，随心打造专属视觉风格。

## 📋 设备与环境要求
在开始之前，请确保您的设备满足以下条件：
*   **系统版本**：Android 8.1 及以上（注：QZMusic 本身要求 Android 10+，故此条通常自动满足）。
*   **权限要求**：设备已获取 **Root 权限** 并激活 **LSPosed 框架**。
    *   *重要提示：由于需要 Hook 系统 UI 以实现歌词显示，Root 和 LSPosed 是必选项。*

## 🛠️ 安装与配置流程
请严格按照以下步骤操作：

1.  **基础准备**：解锁 Bootloader (BL) 并获取 Root 权限。
2.  **环境部署**：刷入 `Zygisk-next` 及 `LSPosed` 模块（相关资源可在本文件夹内找到）。
3.  **安装应用**：安装 `lyricon.apk`（词幕主程序）。
4.  **激活模块**：打开 LSPosed 管理器，勾选并激活“词幕”模块。
5.  **重启设备**：重启手机使模块生效。
6.  **完成配置**：重启后，QZMusic 将自动适配词幕。您可在词幕设置的“歌词提供者”列表中确认是否已识别到 QZMusic，无需额外操作。
